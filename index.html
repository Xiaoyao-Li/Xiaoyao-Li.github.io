Research
GenDexGrasp: Generalizable Dexterous Grasping
Puhao Li*, Tengyu Liu*, Yuyang Li, Yiran Geng, Yixin Zhu, Yaodong Yang, Siyuan Huang
ICRA 2023, Under Review
arXiv / PDF / Project Page / Code
In this study, we introduce GenDexGrasp, a versatile dexterous grasping method that can generalize to out-of-domain robotic hands. We contribute MultiDex, a large-scale synthetic dexterous grasping dataset additionally.
DexGraspNet: A Large-Scale Robotic Dexterous Grasp Dataset for General Objects Based on Simulation
Ruicheng Wang*, Jialiang Zhang*, Jiayi Chen Yinzhen Xu, Puhao Li, Tengyu Liu, He Wang
ICRA 2023, Under Review
arXiv
In this paper, we present a large-scale dexterous grasping dataset, DexGraspNet based on simulation. This dataset features more physical stability and higher diversity than previous datasets.
